# Arşiv — Önceki HTML projesi

Bu klasör, projenin **React Native (Expo) mobil uygulamasına** geçişinden önceki
HTML tabanlı çalışmaları içerir. Referans/tarihçe amaçlı saklanır; aktif geliştirme
artık `../mobile/` altında yürür.

## İçerik
- `KIDS TRADE.dc.html` — ilk web landing + tıklanabilir prototip
- `support.js` — prototip çalışma zamanı
- `index.html` — yönlendirme girişi
- `mobile-app-ui.html` — Material Design 3 mobil UI tasarım mockup'ı (v2)
- `assets/` — prototip görselleri

> Mockup'taki v2 tasarım, `mobile/` altındaki React Native uygulamasına işlenmiştir.
> Render edilmiş ekran görüntüleri için `../screens/` klasörüne bakın.
